## This project is a Student Result Management System developed in C++.
It allows teachers to add, update, search, and save student records.
Students can view their own marks, total, percentage, and grade.
Data is stored permanently using file handling, so records are not lost.
The system uses OOP and vectors to manage multiple student records efficiently.

## Tools used :-
c++ :- The primary programming language for the project.

Library :- iostream, string, vector, fstream, sstream, algorithm, or iomanip.
